# DOS Resident Register Monitor

**DOS Resident Register Monitor** — это резидентная утилита для MS-DOS, написанная на 16-битном x86 ассемблере. 

Программа перехватывает прерывания клавиатуры (`int 09h`) и таймера (`int 08h`). 
По нажатию горячей клавиши она моментально отрисовывает поверх любого окна (прямой записью в видеопамять) менюшку, в котором в реальном времени отображаются **текущие значения всех регистров процессора и флагов**.

Идеально подходит для отладки, изучения работы процессора или просто для рассмотрения красивых рамочек.

## Возможности

- сохраняет участок памяти, над которым рисуется в буфер, при нажатии горячей клавиши снова, восстанавливается вид как ни в чем не бывало
- **Два режима работы**:
  - *Halt-режим (по умолчанию)*: При вызове окна процессор останавливается, замораживая текущую программу. Это бывает полезно, когда хочется посмотреть действия регистров в "моменте"
  - *Динамический режим (`флаг -ed`)*: Окно висит поверх работающих программ, обновляется по таймеру и защищается от воздействия фоновых программ.
- **Большая кастомизация**: Настраивается всё — от координат X/Y до символов отрисовки рамок и цветовых атрибутов текста (через аргументы командной строки).
- **Поддержка типов данных**: Ввод параметров в командную строку поддерживает Hex, а также обычные ASCII символы.

## Как использовать

Всего в командной строке доступно **12** флагов, каждый отвечающий за свою функцию
Каждый флаг начинается с символа '-' затем сразу после идет идентификатор флага.
После этого через пробел указывается значение этого флага. Флаги иможно указывать в любом количестве и в любом порядке. При обнаружении двух одинаковых флагов, программа возьмет самый последний. 
Значение может быть одним из двух:
1) hex-значение: шестнадцатиричное, перед двумя числами должен ставиться префикс H/h (пример: H05)
2) char-значение: ASCII символ, удобен когда нужно указать символ построения рамки. Префикс С/с(пример: С#)

**Флаг стиля** 
это особенный флаг, т.к. его значение - 10 подряд идущих значений. Он сделан для удобства задания комплексной рамки без надобности написания 10 разных флагов с тире. 
Пример использования: (-so H23017867C#0101010101)

**Флаг Режима Работы**
Самый важный флаг. Он укзывается в командной строке без значения, ведь если его просто указать, режим работы рамки сменится с hlt на dynamic

Ниже перечислены все флаги:
  - Х: флаг, позволяющие указать х координату левого верхнего угла рамки
  - Y: флаг, определяющий y координату
  - ft, fs: флаги определяющие буквы, из которых будут состоять соответственно основания и стороны рамки(frame-top, frame-side)
  - lt, rt, rb, lb: флаги уголков (left-top, right-top, right-bottom, left-bottom)
  - cb, ci: флаги, кастомизирующие цвет внутренней части рамки и самой границы рамки.(color-border, color-inner)
  - so: флаг стиля. (style-outer)
  - ed: флаг Режимы работы (enable-dynamic)

## Установка и запуск

Программа компилируется под архитектуру `.286` с использованием модели памяти `tiny` (на выходе получается `.com` файл). Вы можете собрать её с помощью любого компиляторы(проверена совместимость с TASM и MASM).

Запустите скомпилированный файл в среде MS-DOS:
<имя файла>.com -<Имя флага 1> <Значение флага 1> ... -<Имя флага n> <Значение флага n>

После этого намите на кнопку '[' и рамка заработает!
